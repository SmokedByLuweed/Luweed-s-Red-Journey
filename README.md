## Estrutura dos Writeups

Os writeups produzidos até **02 de maio de 2026** referenciam labs pelo índice numérico do [PortSwigger Learning Paths](https://portswigger.net/web-security/learning-paths) (ex: `Lab-31.md`). Essa numeração reflete a ordem dentro do módulo curado, que omite alguns labs disponíveis na plataforma.

A partir de **03 de maio de 2026**, os labs passam a ser referenciados pelo nome descritivo conforme listado em [All Labs](https://portswigger.net/web-security/all-labs) (ex: `sqli-union-oracle-version.md`). Essa mudança garante rastreabilidade direta com o catálogo completo do PortSwigger, sem dependência de numeração de módulo.

Writeups anteriores não foram renomeados para preservar o histórico de commits.
